# Timing Report Summary Example

This example demonstrates analyzing an OpenSTA timing report to extract key metrics and violations.

## Usage

```bash
python examples/timing_report_summary/main.py
```

## Input

The example uses a sample timing report that includes:
- Setup violations
- Hold violations
- Critical path analysis
- Timing slack calculations

## Output

The workflow generates a markdown report summarizing:
- Timing violations found
- Critical paths
- Recommendations for fixing timing issues
